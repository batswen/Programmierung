# Vergleiche
In den meisten Programmiersprachen gibt es den Datentyp "Boolean" bzw "Bool".
Dieser kann nur die Werte "True" oder "False" annehmen.

## Bool'sche Ausdrücke
Es gibt sechs verschiedene Möglichkeiten Werte zu vergleichen:
* == Gleich
* != Ungleich
* > Größer als
* < Kleiner als
* >= Größer oder Gleich
* <= Kleiner oder Gleich
